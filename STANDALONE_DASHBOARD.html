üöÄ How to Access These Dashboards

Note: Claude Code web interface doesn't provide automatic port forwarding. To access the dashboards, follow these steps:

Option 1: Run Locally (Recommended)

  1. Clone the repository to your local machine
  2. Install dependencies: pip install fastapi uvicorn jinja2 pydantic
  3. Start Main Server: python -m uvicorn src.superstandard.api.server:app --port 8080
  4. Start APQC Factory: python apqc_factory_server.py
  5. Open browser: http://localhost:8080/dashboard

Option 2: Use Claude Code Desktop

If you're using Claude Code Desktop (VS Code extension), port forwarding is automatic and the dashboards will be accessible directly.
