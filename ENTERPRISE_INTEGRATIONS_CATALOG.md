## ðŸš€ Usage Examples

### Example 1: Salesforce Integration

```python
from superstandard.integrations.enterprise_api_framework import (
    SalesforceIntegration,
    AuthConfig,
    AuthType
)

# Configure authentication
auth = AuthConfig(
    auth_type=AuthType.OAUTH2,
    credentials={
        "client_id": "your_client_id",
        "client_secret": "your_client_secret",
        "refresh_token": "your_refresh_token"
    }
)

# Create integration
sf = SalesforceIntegration(auth)
await sf.authenticate()

# Use Salesforce
accounts = await sf.query_soql("SELECT Id, Name FROM Account LIMIT 10")
new_opp = await sf.create_opportunity({
    "Name": "New Deal",
    "Amount": 50000,
    "StageName": "Prospecting"
})
```

### Example 2: QuickBooks Integration

```python
from superstandard.integrations.enterprise_api_framework import (

```
