## Instrument Selection

### DRUMS
**Kick Drum**:
- Sample: Heavy 808 kick or synthesize in Operator/3xOsc
- Processing: Boost 60Hz, compress hard (8:1 ratio)
- Pattern: Quarter notes, with syncopation in hook

**808 Bass**:
- Synth: FL Studio 3xOsc / Ableton Operator / Serum
- Root notes in C Minor scale: C, D, Eb, F, G, Ab, Bb
- Pattern: Follow the vocal melody baseline
- FX: Saturate, compress, sidechain to kick

**Snare/Clap**:
- Sample: Layered clap + crisp snare
- Processing: Reverb (plate, 1.2s decay), high-pass at 200Hz
- Pattern: On beats 2 and 4 (classic backbeat)

**Hi-Hats**:
- Sample: Closed hi-hat (trap style)
- Pattern: 1/16 notes with velocity variation
- Rolls: Every 4 bars, use 1/32 triplet rolls

### MELODIC ELEMENTS

**Orchestral Strings** (Intro & Build-ups):
- VST: Native Instruments Symphony Series / BBC Orchestra / Free: Spitfire LABS
- Preset: Legato Strings
- Voicing: Cm chord progression: Cm - Ab - Eb - Bb
- MIDI: Long sustained notes building in layers
- FX: Large hall reverb, slight chorus

**Brass Section** (Chorus):
- VST: Same orchestral library
- Preset: Brass Ensemble / Fanfare
- Pattern: Staccato stabs on downbeats
- Notes: Power chord hits (C + G, Ab + Eb)
- FX: Room reverb, slight delay

**Synth Pads** (Throughout):
- VST: Serum / Massive / Sylenth1 / Free: Vital
- Preset: Warm Analog Pad
- Detuned sawtooth waves
- Filter: Low-pass with gentle resonance
- Chord voicings: Cm9, Ab maj7, Eb add9, Bb sus4

### SPECIAL FX

**Code Sounds** (Background texture):
- Record yourself typing on mechanical keyboard
- Pitch shift down -7 semitones
- Add rhythmic gating synced to 1/16 notes
- Reverb with 100% wet, long decay
- Mix very quietly in background (whisper level)

**Git Commit Sounds** (Transition effects):
- Sample "whoosh" or "success" notification sound
- Reverse it, add reverb
- Use on transitions between sections

## Mixing Guide

### Levels (RMS targets):
- Kick: -12dB
- 808 Bass: -15dB
- Snare: -14dB
- Vocals: -10dB (loudest element)
- Strings: -18dB
- Everything else: Fill in gaps

### EQ Settings:

**Kick**:
- High-pass at 30Hz (remove sub-rumble)
- Boost +4dB at 60Hz (thump)
- Cut -2dB at 300Hz (mud)
- Boost +2dB at 5kHz (click/attack)

**808 Bass**:
- High-pass at 35Hz
- Boost +5dB at 50-60Hz
- Cut at 200-400Hz (muddiness)
- Sidechain to kick (hard ducking)

**Vocals**:
- High-pass at 80Hz
- Boost +3dB at 3kHz (presence)
- De-ess at 6-8kHz
- Slight boost at 10kHz (air)

**Strings**:
- High-pass at 200Hz (make room for bass)
- Gentle boost at 2-4kHz (shimmer)
- Roll off above 12kHz

**Brass**:
- High-pass at 150Hz
- Slight cut at 500Hz
- Boost +2dB at 1-2kHz (bite)

### Compression:

**Master Bus**:
- Ratio: 2:1
- Attack: 30ms
- Release: Auto
- Threshold: -6dB
- Makeup gain: +2-3dB

**Vocals**:
- Ratio: 4:1
- Attack: 5ms
- Release: 40ms
- Threshold: Compress peaks by 3-6dB

**Drums**:
- Parallel compression
- Send to bus with heavy compression (8:1)
- Blend 30% compressed signal

### Effects Chains:

**Vocal Chain**:
1. EQ (corrective)
2. De-esser
3. Compressor (main)
4. EQ (creative)
5. Slight saturation/exciter
6. Send to reverb (plate, 1.5s)
7. Send to delay (1/4 note, 25% feedback)

**Drum Bus**:
1. Transient shaper (enhance attack)
2. Bus compressor (glue)
3. EQ (high-pass at 30Hz)
4. Saturation (warmth)

**Master Chain**:
1. EQ (subtle corrections)
2. Multi-band compressor (gentle)
3. Stereo widener (mids mono, sides wide)
4. Limiter (ceiling at -0.3dB, no more than 3dB gain reduction)

## Export Settings

**For Streaming (Spotify, Apple Music)**:
- Format: WAV
- Bit Depth: 24-bit
- Sample Rate: 48kHz
- Target LUFS: -14 LUFS (streaming standard)
- True Peak: -1.0 dBTP

**For YouTube**:
- Same as above but can go to -13 LUFS

**For Soundcloud**:
- MP3, 320kbps
- -14 LUFS

# ðŸŽ›ï¸ GENERAL PRODUCTION TIPS

## Vocal Recording

### Equipment (Minimum)
- **Mic**: USB condenser (Blue Yeti) or XLR (Audio-Technica AT2020)
- **Interface**: Focusrite Scarlett 2i2 (if XLR)
- **Headphones**: Closed-back (Audio-Technica M50x)
- **Pop Filter**: Essential for "P" and "B" sounds

### Recording Environment
- Quiet room
- Hang blankets on walls (DIY acoustic treatment)
- Record away from walls
- No fans/AC during recording

### Recording Process
1. **Warm up**: 10 minutes of vocal exercises
2. **Multiple takes**: Record each verse 3-5 times
3. **Comp the best**: Pick best parts from each take
4. **Double track**: Record same verse twice, pan L/R (slight)
5. **Ad-libs**: Record freestyle, use best moments

### Vocal Processing Chain (Detailed)
```
1. REMOVE NOISE
   - Noise gate (threshold: -40dB)
   - Or manual editing (cut silence)

2. CORRECT EQ
   - High-pass: 80Hz
   - Cut resonance: Use analyzer to find harsh frequencies
   - Typical cuts: 200-400Hz (muddiness), 3kHz (harshness)

3. DE-ESS
   - Target: 6-8kHz
   - Threshold: Catch only sibilance

4. COMPRESS
   - Ratio: 4:1
   - Attack: 5-10ms
   - Release: 40ms
   - Reduce peaks by 3-6dB

5. CREATIVE EQ
   - Boost presence: +3dB at 2-3kHz
   - Add air: +2dB at 10-12kHz

6. SATURATION
   - Subtle analog warmth
   - Tape or tube saturation
   - Mix 10-20%

7. REVERB (Send)
   - Type: Plate
   - Decay: 1.2-1.8s
   - Pre-delay: 20ms
   - Mix: 15-25%

8. DELAY (Send)
   - Type: 1/4 note
   - Feedback: 25-30%
   - Filter: High-pass at 500Hz
   - Mix: 10-20%
```

## Beat-Making Workflow

### 1. Start with Drums
- Get the groove right first
- Loop 4-8 bars
- Make it bounce

### 2. Add Bass
- Follow kick drum pattern
- Root notes on beat 1
- Melodic movement on offbeats

### 3. Layer Melodic Elements
- Chords first (foundation)
- Melody second (hook)
- Textures third (atmosphere)

### 4. Arrange
- Intro -> Verse -> Chorus -> Verse -> Bridge -> Chorus -> Outro
- Create variations (remove elements in verses, add in chorus)
- Build and release tension

### 5. Mix as You Go
- Rough levels from the start
- Don't wait until the end
- EQ to make space

### 6. Final Mix
- Reference professional tracks
- A/B compare frequently
- Take breaks (ear fatigue)

### 7. Master
- Gentle processing
- Aim for streaming standards
- Don't over-compress

# ðŸŽšï¸ MASTERING FOR STREAMING SERVICES

## Target Specifications

### Spotify
- **LUFS**: -14 LUFS integrated
- **True Peak**: -1.0 dBTP
- **Format**: OGG Vorbis 320kbps (they convert)
- **Upload**: WAV or FLAC

### Apple Music
- **LUFS**: -16 LUFS integrated
- **True Peak**: -1.0 dBTP
- **Format**: AAC 256kbps (they convert)
- **Sound Check**: Enabled (normalizes to -16)

### YouTube
- **LUFS**: -13 to -15 LUFS
- **True Peak**: -1.0 dBTP
- **Format**: WAV for upload

### SoundCloud
- **LUFS**: -14 LUFS (they compress heavily)
- **Format**: Can upload MP3 320kbps
- **Note**: Will sound quieter than other platforms

## How to Hit These Targets

### Measure LUFS
- Plugin: Youlean Loudness Meter (free)
- Where: Last plugin on master chain (before limiter)
- Monitor: Integrated LUFS value

### Limiting Strategy
```
1. Insert limiter as last plugin
2. Set ceiling to -0.3dB (safety margin)
3. Adjust threshold until:
   - LUFS meter shows target
   - Usually 2-4dB of gain reduction
4. Enable true peak limiting
5. Check no clipping on peaks
```

### Don't Over-Limit!
- More is not better
- If you need >6dB limiting, mix is not balanced
- Streaming normalizes anyway
- Preserve dynamics

# ðŸ’¡ PRO TIPS FROM THE PROS

## Mixing
1. **Reference frequently**: A/B with similar professional tracks
2. **Take breaks**: Ears fatigue after 45 minutes
3. **Less is more**: Don't over-process
4. **Trust your ears**: Specs are guidelines, not rules
5. **Check in mono**: Reveals phase issues

## Production
1. **Finish songs**: Don't get stuck in loop creation
2. **Impose limitations**: Creativity thrives with constraints
3. **Learn one new thing**: Every session
4. **Collaborate**: Fresh perspectives help
5. **Have fun**: If you're not enjoying it, listeners won't either

## Workflow
1. **Set timer**: 2 hours max per session
2. **Version control**: Save new version each session
3. **Backup**: Cloud storage + external drive
4. **Organize**: Samples, presets, templates
5. **Routine**: Same time daily if possible
