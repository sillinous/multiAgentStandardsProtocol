## Troubleshooting

### Platform Won't Start?

**Check Python:**
```bash
python --version  # Windows
python3 --version  # Linux/Mac
```
Need Python 3.8 or later from https://www.python.org/downloads/

**Check Port:**
- Default port is 8080
- If busy, change in setup wizard or config file

**Check Firewall:**
- Allow Python through firewall
- Allow port 8080

### Can't Connect to Integrations?

**Test in Setup Wizard:**
- Each integration has "Test Connection" button
- Check your API credentials
- Verify API endpoints
- Check network connectivity

### Workflow Not Executing?

**Check:**
- At least one agent in workflow
- Click the üöÄ GO button
- Watch execution panel at bottom
- Check logs for errors
