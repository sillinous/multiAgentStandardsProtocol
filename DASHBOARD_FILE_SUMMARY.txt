USAGE INSTRUCTIONS
================================================================================

Quick Start:
  1. Run: ./start_dashboard.sh (Linux/Mac) or start_dashboard.bat (Windows)
  2. Open browser to: http://localhost:8080
  3. Dashboard loads with real-time data
  4. Monitor 118+ agents across 13 categories

Manual Start:
  1. Install: pip install -r requirements-dashboard.txt
  2. Backend: python dashboard_server.py
  3. Frontend: cd dashboard_frontend && python -m http.server 8080
  4. Open: http://localhost:8080

Configuration:
  - Edit dashboard_config.yaml for customization
  - Restart services to apply changes

API Access:
  - REST API: http://localhost:8765/api/*
  - WebSocket: ws://localhost:8765/ws
  - Docs: http://localhost:8765/docs
